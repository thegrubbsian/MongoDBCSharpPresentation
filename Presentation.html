NoRM

NoRM is a .NET library for interacting with MongoDB; it provides the following:

  • Strongly Typed Collection Abstraction
  • LINQ Provider for MongoDB Queries
  • Connection Pooling
  • High Performance BSON Serializer
  • An Expando Implementation for Dynamic Data
  • Configuration API for Collection/Property Aliasing
  • Map/Reduce
  • DBReferences
  • Type Discrimination

NoRM

Some stuff it doesn't yet support, but will soon:

  • ReplicaSets or ReplicatSet Gaurantees
  • GridFS
  • Geospatial Indexes/Queries

Those brave souls who want these features may use the unstable branch of NoRM.

NoRMatic

NoRMatic is a gooey surgar layer wrapped around NoRM wrapped around MongoDB :) NoRMatic implements the following:

  • An ActiveRecord(ish) pattern
  • Document Versioning
  • SoftDelete
  • Validation
  • Simple Auditing (User, Timestamps)
  • Save and Delete Event Hooks
  • Global Query Filters
  • Simple Query Logging
